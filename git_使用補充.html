  1. 設定帳號

  1. 建立公私鑰
ssh-keygen -t ed25519 -C "your_email@example.com"
# 詢問你公私鑰要存在哪個地方，預設是 /c/Users/you/.ssh/id_ed25519，你也可以自己改路徑，或是直接按 Enter 用預設的
Enter a file in which to save the key (/c/Users/you/.ssh/id_ed25519):[Press enter]
Enter passphrase (empty for no passphrase): [Type a passphrase] # 直接 Enter 即可
Enter same passphrase again: [Type passphrase again] # 直接 Enter 即可

clip < ~/.ssh/id_ed25519.pub
(如果 clip 沒有效果，可以改成 type ~/.ssh/id_ed25519.pub，然後複製起來)
